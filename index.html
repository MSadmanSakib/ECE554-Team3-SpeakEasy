Project Summary

SpeakEasy: Real-Time Speech-to-Text Processing with RISC-V and VPU Acceleration
SpeakEasy utilizes a state-of-the-art speech-to-text (STT) model, such as Vosk or DeepSpeech, to convert live speech into text. The system is powered by a RISC-V CPU alongside a Vector Processing Unit (VPU) to optimize performance. The RISC-V CPU is responsible for overall program control and task management, while the VPU accelerates the vector computations required for STT processing. Additionally, the VPU processes incoming audio signals, transforming them into structured formats such as frames or data bins for efficient model inference. The transcribed text is then transmitted to an external monitor via a serial I/O interface, such as SPART. Real-time transcription systems like SpeakEasy have a wide range of applications, including voice assistants, automated transcription services, and accessibility tools, enhancing communication and inclusivity in various domains.
